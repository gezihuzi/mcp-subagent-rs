# Frontend Landing Page Project Memory

## Product Context

This workspace represents a marketing landing page that needs stronger visual hierarchy and clearer call-to-action structure.

## Refactor Goals

- Preserve existing content while improving readability and visual rhythm.
- Keep implementation simple: semantic HTML, modular CSS, no framework dependency.
- Ensure desktop/mobile rendering is consistent.

## Collaboration Contract

- Research and layout suggestions happen first.
- Build is handled by frontend-focused agent.
- Review validates accessibility basics and copy clarity.
